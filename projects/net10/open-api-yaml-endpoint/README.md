# OpenAPI YAML Endpoint

This sample shows how to serve OpenAPI documents in YAML format via a dedicated endpoint.

Run with `dotnet watch` inside the sample directory.

- `/openapi/v1.json` - OpenAPI JSON format
- `/openapi/v1.yaml` - OpenAPI YAML format
- `/scalar` - Scalar UI for interactive documentation

This sample uses the [Scalar.AspNetCore](https://github.com/scalar/scalar) package.
